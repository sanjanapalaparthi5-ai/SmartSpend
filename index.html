ℹ️ About SmartSpend

SmartSpend is a Student Expense Tracker System developed to help students efficiently manage daily expenses, monitor spending habits, and maintain monthly budgets. The project uses a structured database management approach to organize expense records, categories, and budget information for better financial awareness and smart money management.


Technologies Used: HTML, CSS, JavaScript, MySQL, GitHub
